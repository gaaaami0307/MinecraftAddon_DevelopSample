# MinecraftAddon_DevelopSample

自分用のマイクラ統合版アドオンのサンプルが載ったページです。
色々な種類のエンティティ,アイテム,パーティクル,関数を作っています。
サンプルを一部書き換えるだけで簡単に動作を変えれるようにしています。
一部動作をクラス・配列を使って分けているので、JSONを知らない人には少々複雑な部分もあるかもしれません。
(自分自身も詳しいわけではないのでぐちゃぐちゃな管理方法になってます。)

This is a MinecraftBE Addon pack sample for myself. 
There are many kinds of entities, items, particles, and functions.
It is easy to change action. But, this addon uses class and array with writing ScriptAPI.
So, it may be a little difficult for people who don't know about JSON.
